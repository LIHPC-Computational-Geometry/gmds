Move the nodes from the selection to their provided new position. Additionally projects unto the geometric association of the node, if any, and does not move if the new position provided is incorrect (ie NaN).

Move the nodes from the selection to their provided new position, by increments. Additionally projects unto the geometric association of the node along the way, if any.

Parameters
[in]ANbIterthe displacement will be done by this number of increments
[in]AQualThresholda node will stay in place if the minimum scaled jacobian of the adjacent cells falls below this threshold
[in]ASelectionNodesthe selection of the nodes that will be moved
[in,out]AMeshthe mesh
[in]AVarNodeDestinationthe new prospective positions
[in]AVarNodeGeomAssociationthe geometric association of the nodes, for those which have one
[out]AVarNodeNbMovesthe number of effective displacements for each node (for debug purposes)
