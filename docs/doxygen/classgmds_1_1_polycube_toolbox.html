std::vector< std::vector< TCellID > > PolycubeToolbox::get_faces_of_block ( )

This function return a vector containing the point's TCellIDs of the corner of each faces of the blocs. The TCellID don't change during the algorithm. The bloc on a reloaded mesh will still be valid.
