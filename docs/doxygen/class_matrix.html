template<int TRow, int TCol, typename TType >
VectorND< 3, TType > Matrix< TRow, TCol, TType >::eulerAngles ( const int &  AR0,
const int &  AR1,
const int &  AR2 
) const

Returns the Euler-angles of the 3x3 rotation matrix *this from the parameters (AR0,AR1,AR2) where each parameter defines a rotation axis as an integer in {0,1,2}. 0 means X, 1 means Y and 2 means Z. So writing.

M.eulerAngles(0, 2, 1);

means to perform the X rotation then the Z rotation and finally the Y rotation. Warning this function must only be used for 3x3 rotation matrix in practice. The implementation comes from Graphics Gems IV.

Parameters
[in]AR0First rotation axis
[in]AR1Second rotation axis
[in]AR2Third rotation axis
Returns
A vector of corresponding rotation angles. Component i corresponds to the i^th parameter
