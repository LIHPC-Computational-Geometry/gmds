Detailed Description

Local Structure to store hex corner data for each input point. A corner data is made of the index of the current point (p), the index of the 3 points helping to form the corner (adj). Each "edge" [p,adj[i]] is geometrically defined by vec[i].

The free field indicates if the corner is already used for a hex. The index field give the position of *this in the list of corners defined in point this->p
