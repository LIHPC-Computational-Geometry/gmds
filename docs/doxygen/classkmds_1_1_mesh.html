◆ createConnectivity()

Connectivity * Mesh::createConnectivity ( const EMeshDefinition  AD)

Create a new m_connectivity, which is associated to the cells of this mesh. The connectivity can be:

  • Downward: R2F, R2E, F2E,
  • Upward: N2R,N2F,N2E,E2R,E2F, F2R,
  • Dim equal: R2R, F2F, E2E, N2N Raise an exception if the connectivity cannot be created
Parameters
ADthe type of connectivity to create.
Returns
A pointer on the connectivity
