Detailed Description

Starting from a frame field, this algorithm generate a set of points that will be used for creating a quasi-structured mesh afterward ( mostly made of quadrilaterals and hexahedrals)

Transport a point from the parameter space to the physical one. This transport is done locally to the linear map associated to a simplex, here we compute the barycentric coordinates of the tetrahedral element ATetParam living in the parametric space is transported to the tet ATetPhys living in the physical space.

Parameters

[in]ATetParamthe tetrahedron in parametric space, defined by four points
[out]ACoordthe barycentric coordinate of AP in ATet if AP is in ATet and there is no numerical computation issue.
Returns
true if APntParam lives in ATetParam and can be transported, false otherwise
