Detailed Description

This class keep the classification link between a mesh of type Mesh and a geom model GeomManager.

Node, edge and face classification are handled since they are mandatory for many algorithms.

◆ writeVTKDebugMesh()

void GeomMeshLinker::writeVTKDebugMesh ( const std::string  AFileName)

For debug purpose, we write an output mesh with only the classified cells (nodes, edge, faces) that are linked to points, curves or surfaces. Each cell type will have two exported data variable, which are classif_entity_dim and classify_entity_id.

As the exported mesh is an extraction of the initial mesh, we also keep the reference id of each cell Warning, the file must be a vtk file (.vtk)

Parameters
AFileNamefilename to store the mesh in vtk format
